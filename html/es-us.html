Versiones de seguridad de Apple

En este documento, se enumeran las actualizaciones de seguridad de los distintos software de Apple.

Acerca de las versiones de seguridad de Apple

Con el fin de proteger a nuestros clientes, Apple no revelará, discutirá ni confirmará problemas de seguridad hasta que se haya llevado a cabo una investigación y estén disponibles las correcciones o versiones correspondientes. Este documento enumera las versiones recientes, incluidas las actualizaciones de seguridad.

Si necesitas soporte técnico para resolver un problema de seguridad (por ejemplo, restablecer la contraseña de tu cuenta de Apple o revisar un cargo reciente del App Store), consulta el artículo Obtener ayuda para solucionar problemas de seguridad.

Si crees que detectaste una vulnerabilidad de seguridad o privacidad en un producto Apple, obtén información sobre cómo enviar un informe.

Obtener las actualizaciones de software más recientes de Apple

Para conservar la seguridad de los productos Apple, es muy importante mantener actualizado el software.

  • La versión más reciente de iOS y iPadOS es 26.2. Obtén información sobre cómo actualizar el software del iPhone o iPad.

  • La versión más reciente de macOS es 26.2. Obtén información sobre cómo actualizar el software de la Mac y cómo permitir actualizaciones en segundo plano importantes.

  • La versión más reciente de tvOS es 26.2. Obtén información sobre cómo actualizar el software del Apple TV.

  • La versión más reciente de watchOS es 26.2. Obtén información sobre cómo actualizar el software del Apple Watch.

  • La versión más reciente de visionOS es 26.2. Obtén información sobre cómo actualizar el software del Apple Vision Pro.

Ten presente que no se pueden deshacer las actualizaciones de software de iOS, iPadOS, tvOS, watchOS y visionOS.
